# Frontend (Next.js)

This is the Next.js (App Router) frontend scaffold for the demo AI-Native SDLC project.

## Routes (placeholders)

- `/` — Diagrams List Page placeholder
- `/editor/[id]` — Editor Page placeholder

## Local run (Docker Compose)

From the repo root:

- `docker compose up --build`

Then open:

- Frontend: http://localhost:3001
- Backend: http://localhost:3000

Navigation:

- From `/`, click “Open example editor” to go to `/editor/demo`
- From `/editor/demo`, click “Back to diagrams list”
